Các kiểu dữ liệu trong js
    1 - Kiểu số ( gọi là number bao gồm cả số nguyên và số thực ) 3 ( số nguyên ) 3.5 ( số thực )
    2 - Kiểu chuỗi ( "hello world" )
    3 - Kiểu boolean ( kiểu chứa 2 giá trị và true và false )
    4 - Kiểu mảng ( danh sách các giá trị )
    5 - Kiểu đối tượng
    6 - Kiểu giá trị null
    7 - Kiểu undefined ( có khai báo biến nhưng không khai báo giá trị )
    Mình có cái từ khóa typeof để kiểm tra kiểu dữ liệu của biến
